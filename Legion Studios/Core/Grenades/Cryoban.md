## Description
Manufactured by Merr-Son Munitions, inc. It was designed to put out fires, could also cause frostbite; nerve damage; and sock to living organisms.

## Features
- Put out ACE Fire sources (e.g. wrecks)
- Can potentially make custom wounds that only affect organics
