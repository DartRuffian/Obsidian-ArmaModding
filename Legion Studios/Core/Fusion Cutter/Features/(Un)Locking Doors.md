#fusion-cutter #features 

Shooting a closed door will start to (un)lock the door depending on its state. (Un)locking an open door wouldn't make sense and would allow players to unlock a door hit with a breaching charge (which locks the door in an open state) which isn't desired. This would use the vanilla door system for best compatibility.

Doors could have a "health" value that would allow certain doors to be harder to slice than others. Should probably just be an object variable
